### 게임 설명
- 유리판을 좌클릭하여 캐릭터를 이동시킬 수 있습니다.
- 눈앞에 놓인 두 종류의 유리는 강화 유리와 일반 유리 하나씩으로 구성되어 있습니다.
- 강화 유리만을 밟아 10번 성공하게 되면 게임이 클리어 됩니다.
- 일반 유리를 밟을 경우 다리 밑으로 떨어지게 되며 탈락합니다.
